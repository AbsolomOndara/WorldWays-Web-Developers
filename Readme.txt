# WorldWays Web-Developers

![WorldWays Logo](assets/img/logo.jpg)

A modern web development agency website showcasing our services, team, and contact information.

## 🔗 Live Demo
[https://worldwaysonedevelopers-wl6ysj1iy-absolom-jasons-projects.vercel.app](https://worldwaysonedevelopers-wl6ysj1iy-absolom-jasons-projects.vercel.app)

## ✨ Features

- **Responsive Design**: Works flawlessly on all devices
- **Interactive Elements**:
  - Animated service cards
  - Video background hero section
  - Smooth scroll navigation
- **Modern Tech Stack**:
  - HTML5, CSS3, JavaScript
  - Bootstrap components
  - AOS (Animate On Scroll) library
- **Contact Integration**:
  - Tawk.to live chat
  - Contact form with validation
